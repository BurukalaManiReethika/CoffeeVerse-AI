# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/CoffeeVerse-AI.git
```

## Open Project

```bash
cd CoffeeVerse-AI
```

## Run

Simply open:

```text
index.html
```

in your browser.

No frameworks.
No build tools.
No dependencies.
